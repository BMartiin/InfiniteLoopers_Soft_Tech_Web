3. Programozási nyelvek és fejlesztő eszközök

  • Programozási nyelvek: Python – kezdőbarát, a projekt nem kíván komplexebb nyelvet
  • IDE: Visual Studio Code – könnyű, elegendő a projekthez
  • Verziókezelő rendszer: GitHub
  • Könyvtárak / frameworkök: Pygame – 2D platformerhez kiváló

4. Tervezett módszerek és technikák

  • Fejlesztési módszertan: Iteratív és inkrementális, heti 1 meeting
  • Tesztelési technikák: Manuális teszt, Exploratory, Playtesting, hibajegyzék, egyszerű unit tesztek
  • Telepítési technikák: Csomagkezelő (pl. pacman, anaconda)
